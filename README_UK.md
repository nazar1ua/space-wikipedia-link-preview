## Як запустити плагін:

1. Заповніть `.env`:
    ```shell
    cp .env.example .env
    ```

2. Змініть дані в `application.conf`, версію в `build.gradle.kts`
і `docker-ci.sh`

3. Запустіть:
    ```shell
    bash docker-ci.sh
    ```
